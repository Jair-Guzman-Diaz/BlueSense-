# BlueSense-
BlueSense_Project 🌿 | Un proyecto Open Source para entusiastas de la acuariofilia y el IoT. El objetivo es crear una aplicación móvil para un acuario estable y autónomo usando hardware de bajo costo y protocolos estándar como MQTT. ¡Contribuciones y mejoras en los algoritmos de control son bienvenidas!
